# switchysharp-optionbak-merge
cli tool for switchSharp option backup file merging.

## Usage

```
switchy-merge -i ./SwitchyOptions-20200826.bak ./SwitchyOptions-20200824.bak ./SwitchyOptions-20200902.bak -o ./now.bak
```
